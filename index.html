Web Grid

CSS Grid based Grid Framework!

A CSS grid based Grid system for easy web development.

The minified Grid CSS weighs less than a KB.

Simple

Web Grid makes use of Grid CSS which it simple and easy to use.

Lightweight

Web Grid uses no floats and flex. Hence making it one of the lightest CSS Frameworks.

Responsive

Web Grid adopts its size according to the device it is viewed on.
